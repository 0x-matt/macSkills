2.4. Chrome 插件推荐

  • One Click Extension Manager
  • OneTab
  • Vimium
  • Image Downloader
  • Video Downloader professional
  • Momentum
  • Octotree
  • Adbock Plus
  • Google 翻译 

One Click Extension Manager

这是一个管理插件的插件。插件太多，有些又暂时用不到，但是依然处于打开状态，消耗内存，不如试试 One Click Extension Manager 来管理插件。 Chrome Web Store

OneTab

快速收起所有打开的 Tab，Chrome Web Store。

Vimium

从名字就能看出来插件是 vim 党的福音，有了它，网上冲浪时手指再也不用离开键盘，去找鼠标了。Chrome Web Store。

  1. 使用 j/k/h/l (熟悉的 vim 操作) 控制页面滚动，u/d 进行上下翻页。

  2. (Shift) O字母来快捷查找浏览历史。

  3. 最炫的是它能够使用 f 快速标记处所有 html 可交互标签，然后键入相应的字母编号实现快速的交互定位。

Image Downloader

快捷下载当前页面的图片，Chrome Web Store。

Video Downloader professional

快捷下载当前页面的视频，不是所有的视频资源都能下载，一些公益性质或者公开的视频可以轻松下载。Chrome Web Store。

Momentum

美化你的空白 Tab 页面，Chrome Web Store。

Octotree

git专属插件，程序员插件，用目录树的结构查看当前 Github 仓库文件，同样适用于开源中国、coding。Chrome Web Store。

Adbock Plus

拦截页面广告，Chrome Web Store。

Google 翻译 

高效的进行浏览器取词翻译， Chrome Web Store。

本文介绍的 Chrome 插件只是冰山一角，内容包含了我的个人偏好，有更多好玩、有用或者无用的插件，欢迎留意评论。
